Open Projects

I like to contribute projects to the community whenever possible.

RestSMSGateway

REST SMS Gateway allows to change your phone into a powerful SMS Gateway which you can control from a computer. This library aims to improve the usability of the API. My first library published in Pypi. (Unofficial Library)

Simple Python Web Crawler

Project to search for links in web pages recursively. The intention was to apply some prior knowledge to a new idea. Using Docker, MongoDB and Python, the script can capture only internal links, as well as "mine" links from other domains and capture the entire internet (or almost).

jararaca

It's a open source system initially created to manage Grupy-RN events and automate tasks related to it. Created using Python (Django) and JavaScript (React), the application runs in a web browser (desktop or mobile).
